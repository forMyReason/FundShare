# Changelog

约定：**新版本在上、旧版本在下**；合并或发布前在 `## [Unreleased]` 下写清用户可见变更（中文简述即可），发版时把该段标题改成带日期的版本号（例如 `## [0.2.0] - 2026-04-30`）。

## [Unreleased]

- 交易与净值
  - 买入改为仅确认日录入；提交时 `apply_date` 与 `confirm_date` 同值写入。
  - 卖出默认模式为“指定买入批次”，并支持“按净值+份额 / 按金额+份额（自动算净值）”两种录入方式。
  - 交易表隐藏申请日，仅展示确认日；金额与手续费按两位小数展示。
  - 净值曲线增强：买入点按同日聚合（显示当日买入笔数、买入份额、剩余份额），支持显示卖出点开关，清仓买入点显示为空心点。
  - 曲线交互优化：区间选择移动到图下并居中；y 轴按约 4% 步进打格；当所选区间超过基金可用历史时给出提示。
  - 买入点可见性修复：交易页优先使用可用净值数据源，区间内无买入点时给出提示文案。
- 维护界面
  - 维护页聚焦为基金新增、删基金、删交易、清空记录，并新增“一键删除基金及全部记录（危险操作）”。
  - 基金列表与选择项支持从 1 开始的展示序号（仅 UI 层，不改底层主键）。
- 界面与文档
  - 暂时隐藏导出/下载入口（组合导出、交易导出、导入示例下载与模板下载）。
  - 移除“多基金净值对比”模块。
  - 所有表格内容统一左对齐。
